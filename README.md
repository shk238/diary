diary
　Androidで動作するシンプルな日記アプリです。

概要
　睡眠時間、コンディションを数値で記録
　「自己啓発」「昼食」「運動」の有無を記録し、
　別画面で数値データはグラフ化、達成の有無は達成率を表示します。

特徴
　シンプルなUI: 必要最低限の入力、表示機能としています。
　生成AIで作成しています。

開発環境
　Language: Kotlin
　IDE: Android Studio
　Min SDK: 26

セットアップ / 実行方法
　1. このリポジトリをクローンします。
　git clone https://github.com/shk238/diary.git
　2. Android Studioでプロジェクトを開きます。
　3. エミュレータまたは実機を接続し、実行ボタンを押してください。

diary
　A simple diary app for Android.

Overview
　This app tracks daily habits and conditions to help visualize your lifestyle:
　Quantitative Logs: Record sleep duration and physical condition as numerical data.
　Activity Tracking: Log the completion of specific activities like "Self-improvement," "Lunch," and "Exercise."
　Visualization:
　Numerical data is displayed as graphs on a separate screen.
　Activity records are calculated and displayed as achievement rates.

Features
　Minimalist UI: Focused on essential input and display for a clutter-free experience.
　AI-Assisted Development: Developed with the assistance of Generative AI.

Development Environment
　Language: Kotlin
　IDE: Android Studio
　Min SDK: 26

Setup / How to Run
　1. Clone the repository:　git clone https://github.com/shk238/diary.git
　Open the project in Android Studio.
　2. Connect an emulator or a physical device.
　3. Click the "Run" button to build and launch the app.
